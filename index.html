## 概要 / Overview

より良い外観の<textarea>を提供します。

## 要求事項 / Requirement

  • ブラウザがカスタム要素、クラス構文、グリッドレイアウト、Shadow-DOM等の比較的新しい機能をサポートしている必要があります。

## 使い方 / Usage

<lined-textarea>を使うためには、以下のCSSとJSファイルを読み込む必要があります。
適宜<head>や<body>内で読み込んでください。

  • HTMLLinedTextAreaElement.css
  • HTMLLinedTextAreaElement.js

### デフォルト書式 / Default Styles

Code 
<lined-textarea>public class Main {
  public static void main (String[] args) {
    System.out.println("Hello, world");
  }
}</lined-textarea>
Result public class Main { public static void main (String[] args) { System.out.println("Hello, world"); } }

### カスタム書式 / Custom Styles

style属性やCSSで背景色、文字色、線色を変更することが可能です。

Code 
<lined-textarea style="
    background-color: ghostwhite;
    border-color: darkslategray;
    color: darkslategray">public class Main {
  public static void main (String[] args) {
    System.out.println("Hello, world");
  }
}</lined-textarea>
Result public class Main { public static void main (String[] args) { System.out.println("Hello, world"); } }
